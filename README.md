# Screel

Gamble your screen time. Set a daily minute bank, link Screen Time, then play **Blackjack** or **Roulette** to win or lose minutes.

## Run

```bash
cd screel
npm install
npm run dev
```

Open the local URL Vite prints (usually `http://localhost:5173`).

## What’s inside

- **2s loading screen** — brand boot into the app
- **Home** — minute bank, streak, XP, daily challenges, quick play
- **Play** — full Blackjack + Roulette tables with minute wagers
- **Bank** — Screen Time connect (demo), daily ceiling, claim rewards, reset day
- **Stats** — net minutes, win rate, hand history
- **You** — profile, toggles, how it works

Progress saves in `localStorage`.
